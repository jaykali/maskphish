# MaskPhish
It is a simple Bash Script to hide phishing URL under a normal looking URL(google.com or facebook.com).

## Installation 

```
git clone https://github.com/jaykali/maskphish
cd maskphish
bash maskphish.sh
```
(Tested on Kali Linux, Termux & Ubuntu)
